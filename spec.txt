Vcard generator
By: Allen K
13.11.2023

Objective:
	Generates a Vcard based on the csv file data.

Solution:
	Python program that automatically takes the data from the csv file and saves it to a new directory as separate files.
	Each file will be created once the python file name followed by the csv file name is executed.
Result:
	A new directory with files containing the vcard data.
	Each file will be of the format: lastname_firstname.txt
Test:
	Test checks if:
	1) A file is created in the given directory
	2) The contents of each file is as per the given vcard format.
